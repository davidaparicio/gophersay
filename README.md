# gophersay
                                                                    
Welcome in GopherSay!

GopherSay is inspired by Cowsay program.

GopherSay allow you to display a message said by a cute random Gopher.

## Pre-requisites

Install Go in 1.16 version minimum.

## Build the app

`$ go build -o bin/gophersay main.go`

or

`$ task build`

## Run the app

`$ ./bin/gophersay`

or

`$ task run`
